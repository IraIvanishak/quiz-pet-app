# Quiz Pet App

## Main Application

The main application serves as a REST API to interact with the quiz data.

### API Endpoints

- **GET /**: Retrieve previews of all quizzes.
- **GET /test?id=<quiz_id>**: Retrieve a specific quiz by its ID.

## Getting Started

Run the application:
```bash
docker-compose up --build
```
